1
Domain Name (Zaruri)
🌐 Domain Name
godaddy.com  |  namecheap.com
₹800–1,500/year
  • 1
    godaddy.com ya namecheap.com par jaayein
  • 2
    Search box mein apna pasand ka naam type karein — jaise sellkaro.in ya apnamarket.com
  • 3
    Domain select karein aur cart mein add karein — .in (~₹500/yr) ya .com (~₹1,000/yr)
  • 4
    Account banayein, payment karein — domain aapka ho jaayega
  • 5
    Auto-renew ON rakhein taaki domain expire na ho

📤 Developer ko ye share karein:

Domain name (e.g. sellkaro.com)
Domain provider ka login access (ya DNS management access)
2
AWS Account — Server + Storage (Zaruri)
☁️ Amazon Web Services (AWS)
aws.amazon.com
Free Tier — 12 Months
  • 1
    aws.amazon.com par jaayein → "Create an AWS Account" click karein
  • 2
    Email address, password aur account name fill karein
  • 3
    Credit/Debit card add karein — free tier mein charge nahi hoga. Sirf verification ke liye ₹2 temporary hold hoga jo 3-5 din mein wapas aa jaata hai
  • 4
    Phone number se OTP verify karein
  • 5
    Support plan mein "Basic support — Free" select karein
  • 6
    AWS Console mein login karein — account ready!
ℹ️ Free Tier mein kya milta hai (12 mahine):
• EC2 t2.micro — App Server (750 hrs/month = poora mahina free)
• S3 — File Storage 5GB free
• CloudFront — Website CDN 50GB/month free
• SES — 62,000 emails/month free (EC2 se)

📤 Developer ko ye share karein:

AWS account ka root email aur password (temporarily)
Ya developer IAM user khud bana lenge aur aapko credentials denge
3
Database — Supabase (Zaruri)
🗄️ Supabase
supabase.com
Free Forever
  • 1
    supabase.com par jaayein → "Start your project" click karein
  • 2
    GitHub account se ya email se sign up karein
  • 3
    Dashboard mein "New Project" click karein
  • 4
    Project name dein, ek strong database password set karein (ise save kar lein!), Region: Southeast Asia (Singapore) ya South Asia select karein
  • 5
    "Create new project" — 2 minute mein project ready ho jaayega
  • 6
    Project → Settings → Database → Connection string → URI tab → copy karein

📤 Developer ko ye share karein:

Database Connection String (URI — postgres://... se shuru hoti hai)
Database Password (jo aapne set kiya tha)
4
Email Service — Resend (Zaruri)
📧 Resend
resend.com
Free — 3,000 emails/month
  • 1
    resend.com par jaayein → "Get Started" click karein
  • 2
    Email se sign up karein aur email verify karein
  • 3
    Dashboard → Domains → "Add Domain" → apna domain daalein (e.g. sellkaro.com)
  • 4
    Resend kuch DNS records dikhayega — screenshot le lein aur developer ko send karein. Developer ye records aapke domain mein add kar denge
  • 5
    Domain verify hone ke baad → API Keys → "Create API Key" → naam dein → Copy karein

📤 Developer ko ye share karein:

Resend API Key (re_ se shuru hoti hai)
Email FROM address — e.g. noreply@sellkaro.com
DNS records ka screenshot (domain verification ke liye)
5
Google Login — Google OAuth (Optional)
🔐 Google Cloud Console
console.cloud.google.com
Bilkul Free
  • 1
    console.cloud.google.com par jaayein (apne Gmail account se login karein)
  • 2
    Top par "New Project" click karein → Project naam dein → Create
  • 3
    Left menu → APIs & Services → OAuth consent screen → External select karein → App name, support email fill karein → Save
  • 4
    Credentials → "+ Create Credentials" → OAuth Client ID → Application type: Web application
  • 5
    Create karne ke baad Client ID copy karein

📤 Developer ko ye share karein:

Google OAuth Client ID (numbers aur letters ka lamba string)
